# ❤️ Heart Disease Prediction System


### A Machine Learning based web application that predicts the likelihood of heart disease based on patient medical parameters. This project helps demonstrate how machine learning can assist in early detection of cardiovascular diseases.

### This project uses Machine Learning algorithms to analyze medical data and predict whether a person is likely to have heart disease based on several health indicators.


# 🚀 Features
### 1) Predicts the probability of heart disease
### 2) Simple and user-friendly interface
### 3) Machine Learning based prediction system
### 4) Data preprocessing and model training
### 5) Accurate predictions using trained ML models
### 6) Fast and efficient prediction system

# 🧠 Machine Learning Workflow
### 1) Data Collection
### 2) Data PreprocessingFeature Selection
### 3) Model Training
### 4) Model Evaluation
### 5) Prediction System Development

# 📊 Input Features
### The model uses several medical parameters such as:
### 1) Age
### 2) Sex
### 3) Chest Pain Type
### 4) Resting Blood Pressure
### 5) Cholesterol
### 6) Fasting Blood Sugar
### 7) Resting ECG Results
### 8) Maximum Heart Rate Achieved

### 6)Machine Learning Algorithms
